# DutchTreat

DutchTreat is a simple gem for splitting the expense among people during a group trip.

## Installation

```ruby
gem 'dutch_treat', git: 'git@github.com:42races/dutchtreat.git'
```

And then execute:

    $ bundle

## Usage

clone the repo and run the tests see the examples

    $ rake test
